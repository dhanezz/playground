## ANSI COLOR CODES

Summary of Common ANSI Color Codes:

- `\033[0;30m`: Black
  - `\033[1;30m`: Black but bold, because of the 1
- `\033[0;31m`: Red
- `\033[0;32m`: Green
- `\033[0;33m`: Yellow
- `\033[0;34m`: Blue
- `\033[0;35m`: Magenta
- `\033[0;36m`: Cyan
- `\033[0;37m`: White
- `\033[1;31m`: Bold Red
- `\033[1;32m`: Bold Green

To reset the color `\033[0m.`
